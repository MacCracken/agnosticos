# jantu

> **Sanskrit**: जन्तु (creature, living being)

**Ethology and creature behavior engine** — instinct, survival drives, social dynamics, swarm intelligence, and animal cognition.

- **Type**: Shared library crate
- **License**: GPL-3.0
- **Version**: 1.0.0
- **crates.io**: [jantu](https://crates.io/crates/jantu)
- **Repository**: [MacCracken/jantu](https://github.com/MacCracken/jantu)

## Relationship to bhava

| | bhava | jantu |
|---|-------|-------|
| **Domain** | Human psyche — emotion, personality, mood, social cognition | Animal brain — instinct, stimulus-response, pack behavior, survival |
| **Model** | OCC appraisal, PAD vectors, Big Five, ACT-R | Ethological drives, fixed action patterns, swarm rules |
| **Cognition** | Reflective, self-aware | Reactive, non-reflective |
| **Consumers** | Agent personalities, NPC dialogue, UI adaptation | Creature AI, ecosystem simulation, flock/herd/swarm behavior |

## Consumers

- **kiran** / **joshua** — creature AI, wildlife simulation, swarm enemies
- **bhava** — cross-reference for hybrid human-animal behavior (e.g., fight-or-flight)
- **impetus** — physics-driven creature locomotion
