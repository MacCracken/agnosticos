| Field | Value |
|-------|-------|
| Status | Stable |

---

## What It Does

- Probability distributions: Normal, Uniform, Exponential, Poisson, Binomial, Bernoulli, Gamma, Beta, Chi-Squared, Student-t, F, Cauchy, Weibull, Multivariate Normal
- Descriptive statistics: mean, standard deviation, KDE, correlation matrix, PCA
- Hypothesis testing: t-tests (one-sample, two-sample), chi-squared, confidence intervals
- Regression: linear, polynomial, and logistic
- Bayesian inference and naive Bayes classification
- Combinatorics: factorials, permutations, combinations, Stirling approximation
- Monte Carlo integration, Metropolis-Hastings MCMC, Gibbs sampling
- Markov chains, Hidden Markov Models (Forward, Viterbi, Baum-Welch)
- Time series: moving average, exponential smoothing, autocorrelation, ARIMA
- Built on hisab for mathematical primitives
- Optional AI features via hoosh integration

## Consumers

- **bhava** — emotion engine (probabilistic mood/personality modeling)
- **phylax** — threat detection (statistical anomaly analysis)
- **daimon** — agent runtime (anomaly detection baselines)
