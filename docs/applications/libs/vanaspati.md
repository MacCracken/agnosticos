# vanaspati

> **Sanskrit**: वनस्पति (lord of the forest, plant/tree)

**Botany and plant behavior engine** — growth simulation, photosynthesis, root systems, seasonal cycles, pollination, and ecosystem dynamics.

- **Type**: Shared library crate
- **License**: GPL-3.0
- **Version**: 1.0.0
- **crates.io**: [vanaspati](https://crates.io/crates/vanaspati)
- **Repository**: [MacCracken/vanaspati](https://github.com/MacCracken/vanaspati)
