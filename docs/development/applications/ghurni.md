# ghurni

> Sanskrit: घूर्णि — rotation / spinning

Mechanical sound synthesis: engines, gears, motors, turbines, clocks. All sounds driven by rotational physics.

- **Version**: 0.1.0
- **Repository**: [github.com/MacCracken/ghurni](https://github.com/MacCracken/ghurni)
- **Depends on**: hisab 1.2
- **Consumers**: kiran, joshua, dhvani

## Key Capabilities

- Engine: 4 types (Gasoline, Diesel, TwoStroke, Hybrid), 1-16 cylinders, RPM-driven firing
- Gear: 4 materials (Steel, CastIron, Brass, Nylon), tooth mesh frequency, resonant ringing
- Motor: 4 types (DcBrushed, AcInduction, Brushless, Servo), EM hum + harmonics
- Turbine: Configurable blade count, blade pass frequency, whoosh, duct resonance
- Clock: 4 types (Wristwatch, WallClock, GrandfatherClock, PocketWatch), escapement tick
- ~1,000x real-time
