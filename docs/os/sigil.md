# Sigil — Trust Verification & Crypto

- **Version**: 1.0.0 (stable)
- **Repo**: [MacCracken/sigil](https://github.com/MacCracken/sigil)
- **License**: GPL-3.0-only
- **Tests**: 142
- **Role**: Single crypto/trust boundary for AGNOS

Ed25519 signing, SHA-256/512 integrity, publisher keyring, key rotation, batch verification, caching, key pinning, hierarchical trust delegation, cross-signing, CRL distribution, audit logging. PQC (ML-DSA/ML-KEM) planned as post-v1.0 feature.

**Consumers**: daimon, kavach, ark, aegis, phylax, mela, stiva, argonaut
